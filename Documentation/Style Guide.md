# Style Guide

## Bootstrap 5 Breakpoints
* Extra small (xs): <576px
* Small (sm): ≥576px
* Medium (md): ≥768px
* Large (lg): ≥992px
* Extra large (xl): ≥1200px
* Extra extra large (xxl): ≥1400px

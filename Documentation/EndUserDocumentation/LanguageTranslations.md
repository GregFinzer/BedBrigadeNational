# 🌐 Language Translations

## ❓ What Is Language Translation?

Language Translation lets website visitors view Bed Brigade content in a preferred language 🌎 (for example, English or Spanish) when translations are available.

This helps families 👨‍👩‍👧‍👦, volunteers 🙋, and community members read website content in the language that is easiest for them.

---

## 👥 Who Uses This Feature?

* 🌍 Public website visitors who want to read pages in a different language
* 🤝 Staff and coordinators who support families in multiple languages
* 🛠️ Administrators who update page content and want those updates translated

---

## 🔄 Changing the Website Language (All Users)

Use these steps on the public website:

1. 🌐 Look for the globe icon and language dropdown in the page header
2. ⬇️ Open the dropdown
3. 🗣️ Select your preferred language
4. ⏳ Wait a moment for the page content to refresh

After you choose a language, the website remembers your selection 💾 for future visits on the same browser.

---

## 🔤 What Gets Translated?

In general, these items are translated when available:

* 🏷️ Page labels and common website text
* 📄 Page content such as headers, stories, and blog/news content
* 📅 Event names and some other dynamic text

If a translation is not available for a specific item yet, the website may show the default English text 🇺🇸.

---

## ⚙️ Language Behavior You May Notice

* 🧭 On your first visit, the website may try to match your browser language
* ❔ If no match is found, the site continues in its default language
* 🔄 Changing language updates content on the current page and other pages you visit

---

## 🛠️ For Administrators: Updating Content for Translation

When administrators edit page content in **Manage Pages**:

* 💾 **Save Only** saves your page changes
* 🔁 **Save and Close** saves your changes and also queues the content for translation

This means translation updates are processed after your content is saved.

---

### 📊 Checking Translation Progress (National Admin)

National Admin users can open **Translation Queue** to monitor translation activity.

Common fields include:

* 📌 **Status** — Current state of each translation item
* 🌐 **Culture** — Language/culture code (example: `es-MX`)
* 🕒 **Queued / Sent / Locked** — Processing timestamps
* ⚠️ **Failure Message** — Error details if a translation did not complete

---

## 💡 Best Practices for Non-Technical Users

* 🧩 Keep language selections simple and consistent when helping someone complete a form
* 👀 After changing language, review important pages to confirm they display as expected
* 🔁 If you update public content and need translation updates, use **Save and Close**
* 🛠️ If translated text looks outdated, ask a National Admin to review the Translation Queue

---

## 🧯 Troubleshooting

If translation does not look right:

1. 🔄 Refresh the page
2. 🌐 Re-select your language in the dropdown
3. 📄 Check another page to confirm whether the issue is page-specific
4. 💾 Confirm admin changes were saved using **Save and Close**
5. 📞 Contact your site administrator if the issue continues

---

## 📌 Summary

Language Translation helps make Bed Brigade content easier to access for more families 👨‍👩‍👧‍👦 and volunteers 🙋. Visitors can switch language 🌐 from the header, and administrators can queue updated content for translation as part of their normal page editing workflow.

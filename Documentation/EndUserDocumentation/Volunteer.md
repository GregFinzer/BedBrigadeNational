# Volunteers

## What Is a Volunteer?

A Volunteer is a person who signs up to help with Bed Brigade events, such as bed builds and bed deliveries.

The system keeps each volunteer's contact details and event sign-ups so local teams can communicate clearly and plan events safely.

---

## How Volunteer Sign-Up Works

Volunteers can register on the public Volunteer page.

### Basic Steps

1. Select a location.
2. Choose an available event.
3. Enter volunteer information.
4. Complete the CAPTCHA check.
5. Submit the form.

If the registration is successful, the volunteer sees a confirmation message on screen.

---

## Information Collected During Sign-Up

### Required Information

| Field | Required | Notes |
|-------|----------|-------|
| First Name | Yes | Up to 20 characters |
| Last Name | Yes | Up to 25 characters |
| Email Address | Yes | Must be a valid email format |
| Phone Number | Yes | Format: (000) 000-0000 |
| Number of Volunteers | Yes | Includes the person signing up; range 1-100 |
| Event Selection | Yes | Volunteer must choose an available event |
| CAPTCHA | Yes | Required to submit |

### Optional Information

| Field | Required | Notes |
|-------|----------|-------|
| Organization | No | Optional group or employer name |
| Message/Notes | No | Up to 4,000 characters |
| Other Languages Spoken | No | Helps with language support |
| Translation Ability | No | Asked when languages are selected |
| Church Attendance | No | Optional |
| Church Name | Conditional | Entered only when church attendance is selected |
| Subscribe to Email | No | Opt-in setting |
| Subscribe to Text | No | Opt-in setting |
| Vehicle Type | No | Used mainly for delivery planning |
| Vehicle Description | Conditional | Entered only when vehicle type is "Other" |

---

## What Happens After a Volunteer Submits

When a volunteer submits the form:

- The system checks for an existing volunteer record by email or phone.
- If found, the existing volunteer profile is updated.
- If not found, a new volunteer profile is created.
- The volunteer is added to the selected event.
- A confirmation email is sent.
- Reminder messages are queued (email and text, when available).

This helps avoid duplicate records and keeps event rosters current.

---

## Updating an Existing Sign-Up

If the volunteer is already signed up for the selected event, the system updates the existing sign-up instead of creating a second one.

Common updates include:

- Number of volunteers attending
- Vehicle details
- Sign-up notes

A new confirmation email is sent after the update.

---

## Canceling (Unregistering) from an Event

A volunteer can remove their event sign-up from the Volunteer page.

After unregistering:

- The sign-up is removed from that event.
- A confirmation email is sent.

The volunteer profile itself may still remain in the system for future events.

## Managing Volunteers (Administrators)

Administrators can view and manage volunteers in the administration area.

### Administrative Actions

**For users who can manage volunteers:**

- **Add** - Create a volunteer record manually
- **Edit** - Update volunteer details
- **Delete** - Remove a volunteer record
- **Search** - Find volunteers quickly
- **Reset** - Clear saved filters and grid settings
- **Export** - Download volunteer data as PDF, Excel, or CSV
- **Print** - Print volunteer lists

**For users with view-only permissions:**

- **Search** - Find volunteers
- **Reset** - Clear filters and return to default view

### Typical Volunteer Details Administrators Can Track

| Field | Description |
|-------|-------------|
| Name and Contact Info | Basic volunteer profile data |
| Organization/Group | Affiliated organization or group |
| Previous Volunteer | Whether they have volunteered before |
| Languages and Translation | Communication support capabilities |
| Volunteer Area | Areas such as bed building or delivery |
| Vehicle Information | Vehicle type and optional details |
| Subscriptions | Email and text communication preferences |
| Notes | Internal or signup notes |

---

## Tips for Volunteers

- Use the same email and phone each time you sign up so your information stays connected.
- If plans change, unregister as soon as possible so another volunteer can take your place.
- Include clear notes when your group size, transportation, or availability needs special handling.

---

## Tips for Staff and Coordinators

- Review event counts regularly to monitor volunteer capacity.
- Use exported files for planning and day-of coordination.
- Keep volunteer contact information up to date to improve reminder delivery.
